# Frenchapeau Classroom | GitHub Education 🇫🇷🎓

مرحبًا بكم في الفصل الدراسي الخاص بنا على GitHub Education.  
هذا الفصل يهدف إلى تنظيم تعلّم اللغة الفرنسية من خلال محتوى **Frenchapeau**، وتقديم المهام والأنشطة والمراجعات في بيئة رقمية واضحة ومنظمة.

## ماذا يقدّم Frenchapeau؟
- دروس ودورات لتعلّم الفرنسية.
- شرح مبسط للقواعد.
- بودكاست ومحتوى للاستماع.
- تدريبات تفاعلية ومفردات.
- موارد تعليمية إضافية وتطبيقات مساعدة.

## ماذا سنقدّم في هذا الفصل؟
- واجبات وأنشطة منظمة.
- ملفات شرح ومراجعة.
- تدريبات عملية على الدروس.
- متابعة التقدم والتغذية الراجعة.
- بيئة دراسية رقمية حديثة تشجّع على الالتزام والتفاعل.

## لماذا GitHub Classroom؟
- تنظيم المهام والملفات.
- سهولة تسليم الواجبات.
- متابعة تقدّم الطلاب.
- بيئة احترافية للتعلّم والتعاون.
